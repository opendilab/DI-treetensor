numel¶

Documentation¶

class treetensor.torch.Size(data, *args, **kwargs)[source]¶
numel() → treetensor.common.object.Object[source]¶

Get the numel sum of the sizes in this tree.

Example:

>>> import torch
>>> import treetensor.numpy as ttorch
>>> ttorch.Size({
...     'a': [1, 2],
...     'b': {'x': [3, 2, 4]},
... }).numel()
26

Torch Version Related

This documentation is based on torch.Size.numel in torch v1.9.0+cu102. Its arguments’ arrangements depend on the version of pytorch you installed.

If some arguments listed here are not working properly, please check your pytorch’s version with the following command and find its documentation.

1
python -c 'import torch;print(torch.__version__)'
